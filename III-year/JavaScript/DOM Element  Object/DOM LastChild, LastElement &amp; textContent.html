Definition

LastChild:

The lastChild property returns the last child node of the specified node, as a Node object.

LastElement Child

The firstElementChild property returns the last child element of the specified element.

Difference between two :

The difference between LastChild property and lastElementChild, is that lastChild returns the last child node as an element node, a text node or a comment node (depending on which one's last), while lastElementChild returns the last child node as an element node (ignores text and comment nodes).
