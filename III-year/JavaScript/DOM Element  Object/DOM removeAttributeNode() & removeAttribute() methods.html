removeAttributeNode

The removeAttributeNode() method removes the specified attribute from an element, and returns the removed attribute, as an Attr Node object.
E.G:
var elmnt =document.getElementById('h31');
var att = elmnt.getAttributeNode('class');
elmnt.removeAttributeNode(att);

removeAttribute

The removeAttribute() method removes the specified attribute from an element.

E.G:
document.getElementById('h32').removeAttribute('class');

difference between removeAttributenode and removeAttribute

The difference between removeAttributeNode method and the removeAttribute() method, is that the removeAttribute() method removes the attribute with the specified name, while removeAttributeNode() method removes the specified Attr object. The result will be the same. Also, the removeAttribute() method has no return value, while removeAttributeNode() method returns the removed attribute, as an Attr object.
